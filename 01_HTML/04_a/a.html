HTML5 a 標籤 (超連結) 教學

1. 不同格式的 href 連結

外部網站 (完整網址)： 前往 Google

協定相對路徑 (自動跟隨當前協定)： 前往 維基百科 (使用 // 寫法)

同 Domain 不同路徑 (根目錄相對路徑)： 回 HTML 首頁

同 Domain 同路徑 不同檔案 (相對路徑)： 點我前往 test.html (同資料夾)

2. 常用 target 屬性

目前分頁開啟 (target="_self", 預設值)： 前往 維基百科

新分頁開啟 (target="_blank")： 前往 維基百科 (新分頁)

範例原始碼對照

<!-- 外部網站 -->
<a href="https://www.google.com">前往 Google</a>

<!-- 協定相對路徑 (自動跟隨 http 或 https) -->
<a href="//www.wikipedia.org">前往維基百科</a>

<!-- 根目錄相對路徑 -->
<a href="/01_HTML/index.html">回 HTML 首頁</a>

<!-- 同資料夾相對路徑 -->
<a href="test.html">前往同資料夾檔案</a>

<!-- 新分頁開啟 -->
<a href="url" target="_blank">連結文字</a>
