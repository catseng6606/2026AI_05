常用的區塊元素 (Block-level Elements)

這類元素預設會「獨佔一行」，即便是內容很短，也會佔滿整行的寬度，並強制下一個元素換行。

1. 標題元素 (h1 - h6)

這是 h1 標題

這是 h2 標題

原始碼：

<h1>這是 h1 標題</h1>
<h2>這是 h2 標題</h2>

2. 段落元素 (p)

這是一個完整的段落。區塊元素之間會有預設的上下間距。

這是另一個段落。

原始碼：

<p>這是一個完整的段落。</p>
<p>這是另一個段落。</p>

3. 通用容器 (div)

這是一個 div 容器，常用於排版組合。
另一個 div。

原始碼：

<div>這是一個 div 容器</div>
<div>另一個 div</div>

4. 清單元素 (ul / ol / li)

  • 項目一
  • 項目二

原始碼：

<ul>
    <li>項目一</li>
    <li>項目二</li>
</ul>

5. 水平線 (hr)

原始碼：

<hr>

6. 引用區塊 (blockquote)

這是一個引用區塊，瀏覽器通常會給予縮排呈現。

原始碼：

<blockquote>
    這是一個引用區塊...
</blockquote>

7. 表格元素 (table)

標題 A 標題 B
內容 1 內容 2

原始碼：

<table border="1">
    <tr>
        <th>標題 A</th>
        <td>內容 1</td>
    </tr>
</table>

區塊元素特性總結：

  • 寬度：預設為父容器寬度的 100%。
  • 高度：由內容撐開，或是由 CSS 設定。
  • 排列：垂直排列，一個接一個。
