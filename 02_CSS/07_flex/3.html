Flexbox 第三課：Flex 三劍客屬性

這三個屬性決定了子元素在容器中如何「伸縮」。

1. flex-grow (增長比例)

當容器有「剩餘空間」時，定義子元素分配空間的比例。

1 (grow: 1)
2 (grow: 2)
3 (grow: 1)
/* 比例解釋：總共 1+2+1 = 4 等分 */
/* 項目 2 會分到 2/4 的剩餘空間，所以長得最快 */
.item-2 { flex-grow: 2; }

2. flex-shrink (收縮比例)

當容器空間「不足」時，定義子元素縮小的程度。設定為 0 則完全不縮小。

我不想變小 (shrink: 0)
我可以變小 (shrink: 1)
/* 設定為 0 可以強迫元素保持寬度不換行或不被擠壓 */
.item-1 { flex-shrink: 0; width: 300px; }

3. flex-basis (基準尺寸)

在分配剩餘空間之前的「初始寬度」。它會覆蓋原本的 width 設定。

基準 200px
基準 100px
/* 它類似於 width，但在 Flexbox 運算中優先權更高 */
.item-1 { flex-basis: 200px; }

總結：簡寫屬性 flex

實務上我們常寫成 flex: [grow] [shrink] [basis]

/* 範例：可增長、可收縮、自動基準寬度 */
.item { flex: 1 1 auto; }

/* 範例：不可增長、不可收縮、固定寬度 200px */
.item { flex: 0 0 200px; }

💡 引導思考 (點擊題目查看答案與示範)

Q1: 如果所有的項目都設為 flex: 1，會發生什麼事？

Ans: 所有項目會等比例均分空間。

1
1
1
.item { flex: 1; } /* 均分空間 */
Q2: 在 flex-direction: column 時，flex-basis 控制的是寬度還是高度？

Ans: 控制的是高度。

40px
60px
.container { flex-direction: column; }
.item { flex-basis: 40px; } /* 在垂直軸上設定基準高度 */
