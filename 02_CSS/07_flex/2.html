Flexbox 第二課：水平與垂直對齊

在 Flexbox 中，對齊分為兩個軸向：
1. 主軸 (Main Axis)：使用 justify-content 控制水平方向。
2. 交錯軸 (Cross Axis)：使用 align-items 控制垂直方向。

Q1: 為什麼 stretch 效果在設定了高度的方塊上看不出來？

Ans: 因為明確的高度設定（height）優先權高於 stretch 拉伸。

Q2: 如果容器沒有設定 height，align-items: center 還會有作用嗎？

Ans: 有作用，但因為容器會縮到跟內容一樣高，所以視覺上看不出「置中」的空間。

/* 容器高度 = 最高項目的高度，因此沒有多餘空間可移動 */
