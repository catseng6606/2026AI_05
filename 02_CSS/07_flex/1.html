範例二：垂直排列 (flex-direction: column)

當我們設定 flex-direction: column 時，主軸會從水平轉向為垂直。

範例二原始碼 (CSS)

.container-column {
  display: flex;
  flex-direction: column;   /* 設定排列方向為「垂直」 */
  justify-content: center;  /* 此時垂直方向變成主軸置中 */
  align-items: center;      /* 此時水平方向變成交錯軸置中 */
  min-height: 400px;
}

範例一原始碼 (HTML 結構)

<div class="container">
  <div class="item">項目 1</div>
  <div class="item">項目 2</div>
  <div class="item">項目 3</div>
</div>

關鍵 CSS 設定

.container {
  display: flex;           /* 開啟 Flex 佈局 */
  justify-content: center;  /* 水平方向：置中 */
  align-items: center;      /* 垂直方向：置中 */
  height: 200px;           /* 設定容器高度 */
}

Q3: 為什麼在垂直排列時，justify-content 變成控制上下對齊了？

Ans: 因為它控制的是「主軸方向」，而 column 將主軸轉向了。

flex-direction: column; 
justify-content: center; /* 此時為主軸(垂直)置中 */
