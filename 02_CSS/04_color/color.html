CSS 色彩表示法與透明度教學

在 CSS 中，我們可以使用多種方式來定義顏色，常見的包含 HEX (十六進位)、RGB 以及設定透明度 (Alpha 通道)。

1. HEX 十六進位色碼

使用 # 開頭，後面接著 6 個字元（分別代表 R、G、B 的數值，範圍從 00 到 ff）。

#3498db
<style>
.hex-color {
    background-color: #3498db;
}
</style>

<div class="color-box hex-color">#3498db</div>

2. RGB 表示法

使用 rgb(紅, 綠, 藍)，每個顏色的數值範圍為 0 ~ 255。

rgb(46, 204, 113)
<style>
.rgb-color {
    background-color: rgb(46, 204, 113);
}
</style>

<div class="color-box rgb-color">rgb(46, 204, 113)</div>

3. 包含透明度的表示法 (RGBA)

使用 rgba(紅, 綠, 藍, 透明度)，透明度 (Alpha) 的範圍從 0.0 (完全透明) 到 1.0 (完全不透明)。

rgba(231, 76, 60, 0.5)
<style>
.rgba-color {
    background-color: rgba(231, 76, 60, 0.5); /* 50% 透明度 */
}
</style>

<div class="color-box rgba-color">rgba(231, 76, 60, 0.5)</div>

4. 包含透明度的 HEX 色碼 (HEXA)

在原本的 6 碼 HEX 後面加上 2 碼代表透明度（例如 80 約為 50% 透明度，00 為完全透明，ff 為完全不透明）。

#9b59b680
<style>
.hex-alpha-color {
    background-color: #9b59b680; /* 50% 透明度 */
}
</style>

<div class="color-box hex-alpha-color">#9b59b680</div>
