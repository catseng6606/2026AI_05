6. 總結 - 真實效果示範
float: left
向左漂浮
,
float: right
向右漂浮
+
float: none
不漂浮 (普通流，會填滿空間)
+
clear: both
清除浮動 (強制換行)
⭐
觀察：float: none 的紫色方塊雖然寫在 green 之後，但因為它不漂浮，它會無視漂浮元素的位置（背景會延伸到漂浮元素下方），而 clear: both 則會乖乖排在所有漂浮物下面。
// 實際 CSS 設定參考 
.box-1 { float: left; }   /* 靠左 */
.box-2 { float: right; }  /* 靠右 */
.box-3 { float: none; }   /* 正常排列，不避開浮動，內容會被擠壓 */
.box-4 { clear: both; }   /* 不准左右有浮動，強制移至下方 */
