CSS Float 文繞圖

CSS Float 屬性可以讓元素向左或向右浮動，並且讓後續的內容繞著它排列。這在設計網頁佈局時非常有用，特別是當你想要在文字周圍放置圖片或其他元素時。

1. 基本文繞圖效果 (Float Basic)

這是 float 最原本的用途：讓文字像流水一樣繞著圖片排列。

這是一段較長的文字。當圖片設定了 float: left 之後，它會靠左對齊，而接下來的文字會自動填補右側剩餘的空間。如果文字夠長，它還會流到圖片的下方，形成一個非常自然的環繞效果。這在新聞排版或部落格文章中非常常見。 這是一段較長的文字。當圖片設定了 float: left 之後，它會靠左對齊，而接下來的文字會自動填補右側剩餘的空間。如果文字夠長，它還會流到圖片的下方，形成一個非常自然的環繞效果。 

<div class="box">
  <img src="..." style="float: left;">
  <p>一段夠長的文字內容，會自動繞著圖片排列...</p>
</div>

2. 當文字不足時 (高度塌陷問題)

如果文字內容太少，無法撐開父容器，浮動的圖片就會「溢出」邊框外：

這段文字太短了。
容器 (.box) 無法包住圖片。

3. 加入 clear: both (解決高度塌陷)

在容器底部加入帶有 clear 的元素，強迫容器擴展以包含所有浮動內容：

同樣是很短的文字，
但因為有了 clear，容器被撐開了！ 

<div class="box">
  <img src="..." style="float: left;">
  <p>短文字...</p>
  <div class="clear"></div>
</div>

<style>
  .clear { clear: both; }
</style>

4. 使用 overflow: hidden (另一種簡潔做法)

不需要額外的標籤，只要在父容器設定 overflow: hidden 也能自動包覆浮動內容：

完全不加 <div class="clear">，
只需在父層加上 overflow: hidden 即可解決！ 

<!-- 在父容器直接加上 overflow: hidden 的樣式 -->
<div class="box" style="overflow: hidden;">
  <img src="..." style="float: left;">
  <p>短文字...</p>
</div>
