💡 教學重點：斷點 (Breakpoint)

使用 @media 可以讓我們根據不同的螢幕尺寸切換 Grid 屬性。在 Grid 中，最簡單的方式就是重新定義 grid-template-areas。

手機版 (Mobile First)

/* 預設樣式：單欄排列 */
.container {
  grid-template-areas: 
    "header"
    "main"
    "sidebar"
    "footer";
}

桌機版 (Media Query)

@media screen and (min-width: 768px) {
  .container {
    /* 轉為雙欄，並重新定義區域分布 */
    grid-template-columns: 200px 1fr;
    grid-template-areas: 
      "header  header"
      "sidebar main"
      "footer  footer";
  }
}

🔍 互動思考 (點擊查看答案)

Q1: 為什麼建議先寫手機版（Mobile First）？

Ans: 這樣可以讓程式碼更簡潔。大多數現代網頁在手機上都是單欄排列（Simple Layout），桌機版才需要處理複雜的欄位。先寫簡單的，再用 @media 增加複雜度，邏輯會更清晰。

Q2: 可以在 @media 裡面修改 grid-gap 嗎？

Ans: 當然可以！ 所有的 CSS 屬性都可以在媒體查詢中修改。通常桌機螢幕較大，我們會把 gap 調大一點讓視覺更開闊。

@media (min-width: 768px) {
  .container { gap: 30px; }
}
