CSS 排版重點設定

/* 1. 父容器清除浮動 (避免高度塌陷) */
.keys {
    overflow: hidden;
}

/* 2. 按鍵基本排版：向左浮動與百分比寬度 */
.btn {
    float: left;
    width: 25%; /* 一行 4 個按鍵 */
    box-sizing: border-box; /* 邊框與內距計入總寬度 */
}

/* 3. 寬按鍵設定：佔據兩格寬度 */
.btn-wide {
    width: 50%;
}
