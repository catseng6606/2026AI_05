CSS 變數 (:root) 教學範例

CSS 變數允許我們將常用的數值（如顏色、間距）定義在一個地方，並在整個網站中重複使用。

主題展示卡片

點擊上方的按鈕，觀察此卡片的背景顏色與邊框顏色如何變化。

這就是 CSS 變數的強大之處：一處修改，全站生效。

💡 教學重點：:root 與 var()

1. 如何定義變數？

習慣上我們會定義在 :root 中，這樣全網頁都能讀取到。變數名稱必須以兩個減號 -- 開頭。

:root {
  --primary-color: #3498db;
  --spacing: 15px;
}

2. 如何呼叫變數？

使用 var() 函數來讀取定義好的值。

.btn {
  background-color: var(--primary-color);
  margin: var(--spacing);
}

🔍 互動思考 (點擊查看答案)

Q1: 為什麼變數要定義在 :root 裡面？

Ans: :root 代表的是 HTML 文件的根元素（等同於 html 標籤），但它的優先權更高。定義在這裡可以確保變數被「全局繼承」，讓所有子元素都能使用。

Q2: 如果變數寫錯了或找不到，網頁會壞掉嗎？

Ans: 不會壞掉，但該屬性會失效。你可以提供「備用值」來防止這種情況：

/* 如果 --main-color 找不到，就使用 red */
color: var(--main-color, red);
