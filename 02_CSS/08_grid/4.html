CSS Grid：自適應填充 (Fill vs Fit)

這兩個屬性通常與 minmax() 搭配使用，用來建立不需要 Media Query 就能自動響應的排版。

1. auto-fill (儘量填充)

容器會盡可能塞入最多的格子。即使格子裡「沒有內容」，它也會把空間預留在那裡。

grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));

2. auto-fit (縮放填滿)

容器會把空的格子「折疊」掉，並讓現有的內容去均分填滿剩餘的空間。

項目 2

grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));

💡 引導思考 (點擊題目查看答案與示範)

Q1: 什麼時候該用 auto-fill？什麼時候該用 auto-fit？

Ans:

  • auto-fit (最常用)：適合希望內容永遠「撐滿」一列的情況（如產品清單）。
  • auto-fill：適合希望保持格子大小固定，且不介意右側留白的情況。
Q2: 如果容器寬度縮小到只能放下一格，兩者還有差別嗎？

Ans: 沒有差別。

當空間不足以產生「空網格」時，兩者的行為會完全一致。只有在「有多餘空間可產生新格子」時，區別才會顯現。
