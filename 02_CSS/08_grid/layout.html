CSS Grid 「span 全寬合併」教學

觀念說明

  • span 12 (全寬合併)：當我們的網格系統是 12 欄時，設定 grid-column: span 12 就能讓該元件橫跨一整列。
  • 版面結構：
    • 頁首 (Header)：單一區塊，合併 12 欄。
    • 頁身 (Main)：分為三塊，分別合併 3、6、3 欄，達成三欄式佈局。
    • 頁尾 (Footer)：單一區塊，合併 12 欄。
Header (span 12)
Left Sider (span 3)
Main Content Area (span 6)
Right Sider (span 3)
Footer (span 12)

原始碼參考 (Source Code)

<!-- CSS 關鍵語法 -->
<style>
    .container {
        display: grid;
        grid-template-columns: repeat(12, 1fr);
    }

    /* 頁首、頁尾：合併全寬 */
    .header, .footer {
        grid-column: span 12;
    }

    /* 頁身：三欄式比例 (3+6+3 = 12) */
    .left-sider { grid-column: span 3; }
    .main-content { grid-column: span 6; }
    .right-sider { grid-column: span 3; }
</style>

<!-- HTML 結構 -->
<div class="container">
    <div class="box header">Header</div>

    <div class="box left-sider">Left</div>
    <div class="box main-content">Center</div>
    <div class="box right-sider">Right</div>

    <div class="box footer">Footer</div>
</div>
