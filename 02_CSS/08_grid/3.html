CSS Grid：視覺化區域佈局

grid-template-areas 是 Grid 最強大的功能之一，它允許你像「畫地圖」一樣，直接在 CSS 中定義版型結構。

Header
Sidebar
Main Content
Footer

核心語法拆解

在 CSS 中，我們使用字串來代表網格的每一列：

grid-template-areas: 
    "header header"   /* 第一列：兩個格子都叫 header (橫跨) */
    "sidebar main"    /* 第二列：左邊 sidebar，右邊 main */
    "footer footer";  /* 第三列：兩個格子都叫 footer (橫跨) */

接著，子元素只需要聲明自己屬於哪個區域：

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main-content { grid-area: main; }
.footer { grid-area: footer; }

💡 引導思考 (點擊題目查看答案與示範)

Q1: 如果我想要側邊欄 (Sidebar) 從最頂端一直延伸到底部，該怎麼改？

Ans: 只要把每一列的第一個單字都改成 sidebar 即可。

Side
Head
Main
Foot
grid-template-areas: 
  "sidebar header"
  "sidebar main"
  "sidebar footer";
Q2: 如果某個格子想要「留白」不放任何區域，該怎麼寫？

Ans: 使用一個句號 . 來代表空網格。

(空)
grid-template-areas: 
  "area-A area-B"
  ".      area-C"; /* 句號代表不指定的空格 */
