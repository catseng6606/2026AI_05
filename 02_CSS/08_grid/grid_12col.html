CSS Grid：12 欄網格系統教學範例

💡 觀念重點：什麼是 span？

在 12 欄網格系統中，我們將頁面寬度切割成 12 個相等的垂直欄位。grid-column: span N 的作用是告訴瀏覽器：「這個元件要橫跨多少個單元格」。

  • span 12：橫跨 12 個格子，也就是佔據 100% 的總寬度。
  • span 6：橫跨 6 個格子，也就是佔據 50% 的總寬度。
  • span 3：橫跨 3 個格子，也就是佔據 25% 的總寬度。

優點：比起傳統的百分比計算，Grid 的 span 語法更直覺且易於維護。

📄 原始碼對照 (HTML & CSS)

<!-- HTML 結構 -->
<div class="grid-container">
    <div class="grid-item header">Header</div>
    <div class="grid-item left-sider">Left Sider</div>
    <div class="grid-item main-content">Main Content</div>
    <div class="grid-item right-sider">Right Sider</div>
    <div class="grid-item footer">Footer</div>
</div>

<style>
    /* 1. 設定容器為 12 欄網格 */
    .grid-container {
        display: grid;
        grid-template-columns: repeat(12, 1fr); /* 核心設定 */
        gap: 15px; /* 設定網格間距 */
    }

    /* 2. 使用 grid-column: span 來合併欄位 */
    .header {
        grid-column: span 12; /* 滿寬 */
    }

    .left-sider {
        grid-column: span 3;  /* 3/12 = 25% */
    }

    .main-content {
        grid-column: span 6;  /* 6/12 = 50% */
    }

    .right-sider {
        grid-column: span 3;  /* 3/12 = 25% */
    }

    .footer {
        grid-column: span 12; /* 滿寬 */
    }
</style>
