CSS 排版重點設定

/* 1. 父容器設定為 Grid 網格容器 */
.keys {
    display: grid;
    /* 
       重點解析：grid-template-columns: repeat(4, 1fr);
       - repeat(4, ...): 表示自動重複宣告 4 次。
       - 1fr (fraction): 彈性網格單位，代表「1等份」的可用剩餘空間。
       結合起來即代表：將容器的水平空間等比例劃分為 4 個相同寬度的欄位。
    */
    grid-template-columns: repeat(4, 1fr);
}

/* 2. 按鍵基本排版：自動依照網格單元格排列，無須設定寬度 */
.btn {
    box-sizing: border-box; /* 邊框與內距計入總寬度 */
}

/* 3. 寬按鍵設定：設定橫跨 2 個欄位 */
.btn-wide {
    grid-column: span 2;
}
