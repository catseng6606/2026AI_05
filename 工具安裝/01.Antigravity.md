# Antigravity (AI 程式碼助理)

## 簡介

Antigravity 是一個專為開發者設計的強大的 AI 程式碼助理。它能協助你完成以下任務：

- **自動完成程式碼**：根據上下文預測並填寫程式碼。
- **程式碼生成**：透過自然語言描述生成完整的函式或網頁佈局。
- **錯誤修復**：自動識別程式碼中的 Bug 並提供修復建議。
- **技術問答**：直接在編輯器中詢問編程相關問題。

---

## 下載與安裝

### 1. 下載 Antigravity

由於 Antigravity 是獨立的應用程式（或專屬套件），請依照以下步驟下載：

1. 開啟瀏覽器，前往 Antigravity 的官方下載頁面（或課程提供的下載連結）。
2. 根據你的作業系統（Windows / macOS / Linux）選擇對應的安裝檔進行下載。

### 2. 安裝與設定

**Windows 系統：**

1. 執行下載的 `.exe` 安裝程式。
2. 按照安裝精靈的指示完成安裝，並選擇是否建立桌面捷徑。
3. 啟動 Antigravity 應用程式並進行初始設定（如登入帳號或輸入 API Key）。

**macOS 系統：**

1. 開啟下載的 `.dmg` 檔案。
2. 將 Antigravity 圖示拖曳至「應用程式 (Applications)」資料夾。
3. 到「應用程式」中啟動 Antigravity。

## 常用功能與快捷鍵

- **開啟對話面板**：`Ctrl + Alt + A` (預設快捷鍵，視版本而定)
- **程式碼補全**：在輸入時，按下 `Tab` 鍵即可採納 AI 建議。
- **產生程式碼**：在檔案中輸入 `//` 或選取程式碼後，呼叫對話面板要求 AI 進行修改。

## 為什麼要用 Antigravity？

- **提升效率**：減少重複性的打字工作，讓你專注於邏輯開發。
- **學習幫手**：對於不熟悉的語法或框架，Antigravity 能即時提供正確的範例與解釋。
- **現代開發必備**：AI 輔助開發已成為主流，學會使用 Antigravity 將大大增加你的競爭力。

## 延伸套件

1. 基礎環境類 (讓 Antigravity 更好用)
   - Chinese (Traditional) Language Pack for Visual Studio Code
     - 功能： 這大概是你的首要之選！它會把 Antigravity 的介面 (選單、按鈕等) 全部變成你我熟悉的繁體中文，上手更容易。
   - Material Icon Theme
     - 功能： VS Code 預設的檔案總管中，所有資料夾和檔案圖示都長得差不多。安裝這個套件後，它會根據檔案類型 (例如 .js, .css, .py, .md) 顯示不同、美觀的圖示。
     - 為什麼好用： 你能一眼就分辨出檔案類型，找檔案快非常多！
   - Prettier - Code formatter
     - 功能： 這是目前最受歡迎的「程式碼格式化工具」。它不管你程式碼寫得多亂 (例如縮排亂七八糟、分號亂加)，只要你一存檔，它「唰」的一聲，自動幫你排版得整整齊齊。
     - 為什麼好用： 維持程式碼風格一致性是專業開發者的基本功，它能幫你省下大量手動排版的時間。
   - Remote - SSH
     - 功能： 允許你透過 SSH 連接到遠端伺服器或虛擬機，並直接在遠端環境中編輯、執行與除錯程式碼。
     - 為什麼好用： 能讓你在本地端享受順暢的編輯體驗，同時利用遠端機器的運算資源，非常適合開發需要特定環境或高效能的專案。

2. 網頁開發類
   - Live Server
     - 功能： 如果你是要寫網頁 (HTML/CSS/JavaScript)，這是必裝！它會在你的電腦上啟動一個本地伺服器，並在瀏覽器中打開你的網頁。
     - 為什麼好用： 最棒的是，當你修改 HTML 或 CSS 並存檔時，它會自動重新整理瀏覽器！你不用再手動按 F5 了，修改立刻看得見。
   - ESLint
   - Auto Rename Tag
   - IntelliSense for CSS class names in HTML
   - Git History
